After running seyComplete, you will get an output that can be copy pasted into sheets.

After you copy paste it, you should do
1: keep it highlighted and split it to list (in Data-> Split Text To Columns- should default to split by commas)
2: format delta column to be  Number -> #,##.000
3: format RGB distance to be Number -> 0
4: format Whiteness to be Number -> #,##.00
